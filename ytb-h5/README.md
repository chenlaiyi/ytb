# 净水器管理系统 H5 版本

## 项目简介
这是一个基于 Vue 3 + Vite + Vant 4 开发的净水器设备管理系统 H5 版本。

## 技术栈
- Vue 3
- Vite
- Vant 4 (移动端 UI 组件库)
- Vue Router
- Axios

## 项目结构
```
src/
├── api/          # API 接口
├── components/   # 公共组件
├── router/       # 路由配置
├── stores/       # 状态管理
├── utils/        # 工具函数
├── views/        # 页面组件
└── main.js       # 入口文件
```

## 开发环境启动
```bash
npm install
npm run dev
```

## 构建部署
```bash
npm run build
```
